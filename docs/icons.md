# Extension Icons

The `icons/` directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create these icons using any image editing software. Here are some suggestions:

1. **Design Tools:**
   - Figma (free)
   - Canva (free)
   - Adobe Illustrator/Photoshop
   - GIMP (free)

2. **Icon Requirements:**
   - Square format (16x16, 48x48, 128x128)
   - PNG format with transparency
   - Simple, recognizable design
   - Works well at small sizes

3. **Quick Solution:**
   - Open `create-icons.html` in your browser (in project root)
   - Icons will auto-generate
   - Right-click each canvas and save as the filenames above into the `icons/` folder
   - Or use an online icon generator with text "SMR" or a focus/productivity symbol and export at all three sizes

## Temporary Icons

If you need to test the extension immediately, you can:
1. Create simple colored squares as placeholders
2. Use any 16x16, 48x48, and 128x128 PNG images
3. Replace them later with proper icons before publishing
